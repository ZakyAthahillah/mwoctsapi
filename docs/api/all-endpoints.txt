Access Notes
------------
- Public without token:
  POST /api/register
  POST /api/login
  POST /api/refresh

- Require JWT token:
  POST /api/logout
  GET /api/me
  GET /api/dashboard
  All downtime endpoints
  All job endpoints
  All MTBF endpoints
  All MTTR endpoints
  All FBDT endpoints
  All area endpoints
  All division endpoints
  All group endpoints
  All informant endpoints
  All machine endpoints
  All operation endpoints
  All part endpoints
  All position endpoints
  All reason endpoints
  All serial number endpoints
  All shift endpoints
  All technician endpoints

- Require JWT token and admin middleware:
  All permission endpoints
  PUT /api/users/{user}
  DELETE /api/users/{user}

Positions Examples
------------------
List positions
GET /api/positions?per_page=10&search=posisi&area_id=1

Create position
POST /api/positions
{
  "area_id": 1,
  "name": "Posisi A",
  "description": "Deskripsi posisi",
  "status": 1
}

Update position
PUT /api/positions/{position}
{
  "area_id": 1,
  "name": "Posisi A Update",
  "description": "Deskripsi posisi update",
  "status": 1
}

Delete position
DELETE /api/positions/{position}

Usage Examples
--------------

Common Headers
--------------
Use these headers for authenticated endpoints:

Authorization: Bearer <your_jwt_token>
Accept: application/json
Content-Type: application/json

Authentication Examples
-----------------------
Register
POST /api/register
{
  "name": "Admin User",
  "email": "admin@example.com",
  "username": "adminuser",
  "password": "password",
  "password_confirmation": "password"
}

Login
POST /api/login
{
  "login": "admin@example.com",
  "password": "password"
}

Get authenticated user profile
GET /api/me

Get dashboard summary

Jobs Examples
-------------
List jobs
GET /api/jobs?per_page=10&status=new&machine_id=1

Get job detail
GET /api/jobs/10

Start job
PUT /api/jobs/10/start
{
  "technician_id": 1,
  "processing_date_start": "2026-04-17 09:00:00"
}

Start extended job
PUT /api/jobs/10/start-extend
{
  "technician_id": 1,
  "processing_date_start": "2026-04-17 13:00:00"
}

Extend job
PUT /api/jobs/10/extend
{
  "processing_date_finish": "2026-04-17 15:30:00",
  "notes": "Butuh spare part tambahan"
}

Finish job
PUT /api/jobs/10/finish
{
  "processing_date_finish": "2026-04-17 15:30:00",
  "operation_id_actual": 1,
  "notes": "Pekerjaan selesai",
  "is_change_part": false
}

Approve job
PUT /api/jobs/10/approve
{
  "approved_at": "2026-04-17 16:00:00",
  "approved_by": 5,
  "approved_notes": "Sudah sesuai"
}

FBDTs Examples
--------------
List FBDT years
GET /api/fbdts?per_page=10&area_id=1

Check FBDT year
GET /api/fbdts/check?area_id=1&year=2026

Get FBDT detail
GET /api/fbdts/2026?area_id=1

Create FBDT
POST /api/fbdts
{
  "area_id": 1,
  "year": 2026,
  "targets": [
    {
      "month": 1,
      "fb": 10,
      "dt": 20,
      "mtbf": 30,
      "mttr": 40
    },
    {
      "month": 2,
      "fb": 11,
      "dt": 21,
      "mtbf": 31,
      "mttr": 41
    }
  ]
}

Update FBDT
PUT /api/fbdts/2026
{
  "area_id": 1,
  "targets": [
    {
      "month": 1,
      "fb": 15,
      "dt": 25,
      "mtbf": 35,
      "mttr": 45
    }
  ]
}

Machines Examples
-----------------
List machines
GET /api/machines?per_page=10&search=mesin&area_id=1

Create machine
POST /api/machines
{
  "area_id": 1,
  "code": "MCH001",
  "name": "Mesin Potong",
  "description": "Mesin untuk proses potong",
  "image": "front.png",
  "image_side": "side.png",
  "status": 1
}

Saved values example for string input:
image=images/machines/{id}/front.png
image_side=images/machines/{id}/side.png

Create machine with file upload
POST /api/machines
Headers:
Authorization: Bearer <your_jwt_token>
Accept: application/json
Content-Type: multipart/form-data

Form fields:
area_id=1
code=MCH001
name=Mesin Potong
description=Mesin untuk proses potong
image=<front-image-file>
image_side=<side-image-file>
status=1

Saved values example:
image=images/machines/10075/front.jpeg
image_side=images/machines/10075/side.jpeg

Update machine
PUT /api/machines/{machine}
{
  "area_id": 1,
  "code": "MCH001",
  "name": "Mesin Potong Update",
  "description": "Mesin update",
  "image": "images/machines/10075/front.png",
  "image_side": "images/machines/10075/side.png",
  "status": 1
}

Delete machine
DELETE /api/machines/{machine}
